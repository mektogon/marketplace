# Marketplace
Интернет-площадка для размещения товаров.

## Технологии
Проект разрабатывался с использованием:

- Spring Framework (Core, Boot, Data JPA, HikariCP)
- Lombok
- MapStruct
- PostgreSQL
- Swagger
